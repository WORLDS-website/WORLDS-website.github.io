What is WORLDS?
WORLDS is a game engine that looks like it was made in the 90's, basically having those graphics from late 90's game engines such as the first Unreal Engine that has been released. The reason why this engine exists, is because modern engines are too focused on making the most realistic graphics, modern mainstream game engines are just flexing on each other, on who has the best graphics, while ignoring average PC specs, and only focusing on expensive PC specs.
So that's why this engine is able to run from Windows 95 to Windows 11, because I want my engine to be runnable on every single operating system as possible, Linux is included on the list, I am focused on Windows at the moment. But don't worry for those who use Linux, I will eventually make a Linux port for you.
